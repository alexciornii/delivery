# Delivery
An example of delivery application built using Django.


### Requirements
* python 3.8
* mongoDB

### Run server

#### Local

By default, the server runs on port 8989.
```bash
$ cd backend
$ chmod +x ./bin/run
$ ./bin/run
```

The port can changed by adding it as parameter on running run script.
```bash
$ cd backend
$ chmod +x ./bin/run
$ ./bin/run 8080
```


#### Docker
 
```bash
$ docker-compose up  
```
